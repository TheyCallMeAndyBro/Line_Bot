## Installing

1.Clone the repository
```
git clone https://github.com/TheyCallMeAndyBro/Line_Bot.git
```

2.Move to file
```
cd Line_Bot
```

3.Install npm packages
```
npm install
```

4.Set your LINE API / OPENAI API
```
reference .env.example
```

5.Start
```
npm start
```

6..Stop
```
ctrl +　ｃ
```
